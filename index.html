Universo da Doçura — Custom WordPress Build

A fully custom WordPress application built without page builders — coded from the theme layer up using PHP templates, custom post types, custom queries, MU-plugins, and JavaScript-powered UI. Universo da Doçura is an evolving digital archive of global dessert culture, where each locale, pastry, and artisan is dynamically generated through custom fields and relationships.

Skills: Custom WordPress Theme Development · Custom Post Types (Locale, Pastry Case, Artisan) · Advanced Custom Fields (ACF) · Custom PHP Templates (archive-locale.php, single-locale.php, custom page templates) · MU-plugin architecture for Swiper.js locale hero slider · Custom WP_Query loops · Vanilla JS search overlay & mobile nav · Custom CSS · Mobile-first responsive layout
